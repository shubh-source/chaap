Customer Experience

Step 1: Welcome & Verification

Customer jab table pe rakhay QR code ko scan karta hai, toh ek premium welcome screen dikhta hai CHAAPSHALA ki branding ke saath. Yahan pe:

  • GPS Verification: System check karta hai ki customer restaurant ke 100 meter ke andar hai ya nahi (fake orders se protection)
  • Phone Number: Customer apna 10-digit mobile number daalata hai
  • New Customer: Agar pehli baar aaya hai toh naam bhi maanga jaata hai
  • Returning Customer: Agar pehle aa chuka hai toh seedha welcome message aata hai naam ke saath

Admin Dashboard
Restaurant owner ke liye complete control panel

Admin Dashboard ek powerful control center hai jahan se restaurant ka poora operation manage hota hai — revenue, orders, tables, customers, analytics sab ek jagah.

Dashboard Sections (7 Panels)

Panel Icon Features
Dashboard 📊 Today's Revenue, Total Orders, Pending Payments, Occupied Tables — real-time stat cards. Plus "Needs Attention" table showing orders that need action.
Live Orders 🟢 All active orders with Status (New/Preparing/Ready/Served), Payment status, Action buttons — "Mark Served" aur "Collect Cash"
History 📋 Completed orders list. Monthly PDF Report download (jsPDF). Erase All Orders option for data reset.
Tables 🪑 15-table visual grid map — Free (grey) vs Occupied (gold highlight). Real-time status updates.
Customers 👥 Customer database — Phone, Name, Total Orders, Total Spend. Automatic tracking.
Analytics 📈 Revenue chart (7 days), Peak hours bar chart, Top 5 selling items (doughnut chart), VIP customers (top spenders) with 👑
QR Codes 📱 Generate unique QR codes for each table. Preview on screen. Print all QRs for table placement.

Auto Bill Printing 🖨️

Automatic Thermal Bill Print
Jab bhi koi order PAID hota hai, system automatically thermal receipt style bill print kar deta hai — Order ID, Table, Customer details, Itemized list, Total, Payment method sab included. Duplicate printing ka protection bhi hai.

Monthly PDF Reports 📄

Admin History section mein month select karke ek professional PDF report download kar sakta hai jisme:

  • CHAAPSHALA header with month name
  • Total completed orders count
  • Total revenue for that month
  • Complete order table — ID, Date, Table, Items, Payment Method, Amount

Security Features
Restaurant ki data aur payment security
🔐
JWT Authentication
Admin aur Kitchen login httpOnly cookies mein JWT token store karta hai — 12 hour expiry, browser close pe session khatam.
📍
GPS Geo-Fencing
Customer tabhi order kar sakta hai jab wo restaurant ke 100 meter ke andar ho. Haversine formula se distance calculate hota hai. Fake orders blocked.
💳
Razorpay HMAC Verification
Online payments HMAC-SHA256 cryptographic signature se verify hoti hain. Tamper-proof payment confirmation.
🛡️
Environment Variables
Saari API keys, passwords, database credentials .env file mein secure hain — code mein koi bhi secret hardcoded nahi hai.

Wallet & Cashback System
Customer loyalty program — repeat visits ka secret
💰 5% Cashback on Every Online Payment
Jab bhi customer online pay karta hai (UPI/Card/Wallet), uske account mein 5% cashback credit hota hai. Ye cashback next visit pe bill se kat jaata hai. Isse customers baar baar aate hain!

Wallet Flow:

  • Earn: Customer ₹500 ka order online pay karta hai → ₹25 wallet mein aata hai
  • View: Customer ke menu screen pe 💳 ₹25 wallet balance dikhta hai
  • Use: Next order mein checkout pe "Use Wallet" toggle → bill se ₹25 kat jaata hai
  • Stack: Multiple visits ka cashback accumulate hota hai

Billing Structure

GST 5% of Subtotal ₹25

Setup Guide
System chalane ke liye kya chahiye

Requirements

  • Server: Node.js installed machine (VPS, local PC, ya cloud)
  • Database: PostgreSQL database (free tier available)
  • Payment: Razorpay account (test/live keys)
  • WiFi: Restaurant mein stable internet connection
  • Devices: Kitchen mein ek tablet/monitor, Admin ke liye laptop/PC
